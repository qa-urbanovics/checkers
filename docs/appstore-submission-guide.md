# Публикация в App Store — пошаговая инструкция

Открой appstoreconnect.apple.com → войди → My Apps → твоё приложение.

---

## Шаг 1 — Информация о приложении

Слева нажми **«Информация о приложении»**.

### Раздел «Основная информация»

**Название** (поле Name)
```
Checkers: Emerald Courts
```

**Подзаголовок** (поле Subtitle)
```
Russian & International Draughts
```

**Основная категория** → выбери **«Игры»** → подкатегория **«Настольные»**

**Дополнительная категория** → выбери **«Игры»** → подкатегория **«Стратегии»**

### Раздел «Права на контент»

Вопрос «Содержит ли ваше приложение материалы третьих сторон?» → **«Нет»**

### Раздел «Политика конфиденциальности»

**URL политики конфиденциальности**
```
https://qa-urbanovics.github.io/checkers/privacy.html
```

Нажми **«Сохранить»** вверху страницы.

---

## Шаг 2 — Возрастные ограничения

Слева нажми **«Информация о приложении»** → найди раздел **«Возрастные ограничения»** → нажми **«Редактировать»**.

Все пункты выставь в **«Нет»** или **«Отсутствует»**:

- Мультяшное или фэнтезийное насилие → **Отсутствует**
- Реалистичное насилие → **Отсутствует**
- Сексуальный контент → **Отсутствует**
- Обнажённость → **Отсутствует**
- Употребление алкоголя, табака, наркотиков → **Отсутствует**
- Азартные игры → **Нет**
- Ужасы / Страх → **Отсутствует**
- Нецензурная лексика → **Отсутствует**
- Медицинская тематика → **Отсутствует**
- Все остальные вопросы → **Нет**

Нажми **«Готово»** → рейтинг автоматически станет **4+**.

Нажми **«Сохранить»**.

---

## Шаг 3 — Конфиденциальность приложения

Слева нажми **«Конфиденциальность приложения»** → нажми **«Начать»**.

Вопрос «Собирает ли ваше приложение данные?» → **«Нет, наше приложение не собирает данные»**

Нажми **«Опубликовать»**.

---

## Шаг 4 — Цены и доступность

Слева нажми **«Цены и доступность»**.

- **Цена** → **«Бесплатно (0 руб.)»**
- **Доступность** → оставь все страны (по умолчанию)

Нажми **«Сохранить»**.

## Шаг 7 — Сборка (Build)

В разделе версии найди блок **«Сборка»** → нажми **«+»** → выбери загруженный из Xcode билд.

Появится вопрос **«Использует ли приложение шифрование?»** → **«Нет»**

---

## Шаг 8 — Экспортные требования

Если отдельно спросит про шифрование (Export Compliance):

**Использует ли приложение шифрование?** → **«Нет»**

---

## Шаг 9 — Отправить на проверку

1. Убедись что все разделы слева отмечены галочкой
2. Нажми **«Добавить для проверки»** (правый верхний угол)
3. На вопрос «Приложение предназначено для детей?» → **«Нет»**
4. На вопрос про рекламу → **«Нет»**
5. Нажми **«Отправить на проверку»**

---

## После отправки

| Этап | Срок |
|------|------|
| Обработка билда после загрузки из Xcode | 5–15 минут |
| Проверка Apple | 1–3 дня (обычно ~24 ч) |
| Публикация после одобрения | Автоматически |

Если отклонят — придёт письмо с причиной. Исправляешь → Submit for Review снова.
